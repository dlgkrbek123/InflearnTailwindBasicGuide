=> .scss를 통해 @apply를 사용

<div class="grid-i">hello</div>

.grid-i {
  @apply border-4 border-blue-500;
}

------------------------------------------------------

node로 설치

> npm intall -D tailwind
> npx tailwind init
=> tailwind.config.js 생성

!!input.css에 아래의 사항을 추가!!

.grid-i {
  @apply border-4 border-blue-500;
}

> npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
=> cli로 빌드 실행
